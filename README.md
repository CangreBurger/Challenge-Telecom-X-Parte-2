Challenge Telecom X: Análisis de evasión de clientes – Parte 2
📌 Descripción del proyecto

Este proyecto forma parte del Challenge Telecom X: Análisis de evasión de clientes – Parte 2, cuyo objetivo es desarrollar modelos de Machine Learning capaces de predecir qué clientes tienen mayor probabilidad de cancelar sus servicios.

La evasión de clientes (churn) es un problema crítico para las empresas de telecomunicaciones, ya que implica pérdida de ingresos y aumento en los costos de adquisición de nuevos clientes. Por esta razón, Telecom X busca anticiparse a este comportamiento utilizando modelos predictivos basados en datos históricos de clientes.

En esta segunda parte del desafío, se construye un pipeline de modelado predictivo, partiendo de los datos previamente limpiados y preparados en la Parte 1 del proyecto.

🎯 Objetivos del proyecto

Preparar los datos para el modelado (tratamiento, codificación, normalización).

Realizar análisis de correlación y selección de variables.

Entrenar dos o más modelos de clasificación.

Evaluar el rendimiento de los modelos con métricas.

Interpretar los resultados, incluyendo la importancia de las variables.

Crear una conclusión estratégica señalando los principales factores que influyen en la cancelación.
